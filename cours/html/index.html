Les différentes choses dans le HTML

Le HTML (HyperText Markup Language) est le langage fondamental utilisé pour créer des pages web. Il fournit la structure et le contenu de la page, tandis que le CSS s'occupe de la présentation visuelle et le JavaScript ajoute l'interactivité.

Cette section de notes te permettra de comprendre les différentes balises HTML et comment les utiliser correctement pour structurer tes pages web.

Les trois parties principales du HTML

Partie Description
<!DOCTYPE html> Déclare le type de document et la version du HTML utilisée. Doit toujours être la première ligne de ton fichier HTML.
<head> Contient les métadonnées et les informations invisibles de la page (titre, encodage, liens CSS, scripts, etc.). Consulte la page dédiée au head pour plus de détails.
<body> Contient tout le contenu visible de la page (texte, images, vidéos, liens, etc.). Consulte la page dédiée au body pour plus de détails.
