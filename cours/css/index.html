Résumé des notions importantes

Base du CSS

Élément Utilité Exemple
CSS Permet de styliser une page HTML. Changer les couleurs, tailles et positions.
Sélecteur Sert à choisir l'élément HTML à modifier. p
Propriété Indique ce qu'on veut changer. color
Valeur Précise comment on veut modifier la propriété. blue
Règle CSS Ensemble formé du sélecteur, de la propriété et de la valeur. p { color: blue; }
Commentaire Permet d'écrire une note dans le CSS. /* commentaire */

Les sélecteurs

Sélecteur Utilité Exemple
balise Sélectionne toutes les balises du même type. h1
.classe Sélectionne tous les éléments ayant cette classe. .menu
#id Sélectionne un seul élément unique. #header
* Sélectionne tous les éléments. *
element element Sélectionne un élément à l'intérieur d'un autre. nav a
element:hover Applique un style quand la souris passe dessus. a:hover

Texte

Propriété Utilité Exemple
color Change la couleur du texte. color: black;
font-size Change la taille du texte. font-size: 20px;
font-family Change la typo d'écriture. font-family: Arial, sans-serif;
font-weight Change l'épaisseur du texte. font-weight: bold;
font-style Change le style du texte. font-style: italic;
text-align Aligne le texte. text-align: center;
text-decoration Ajoute ou enlève une décoration au texte. text-decoration: underline;
line-height Change l'espace entre les lignes. line-height: 1.5;
letter-spacing Change l'espace entre les lettres. letter-spacing: 2px;

Couleurs et arrière-plans

Propriété / valeur Utilité Exemple
background-color Change la couleur de fond. background-color: pink;
Nom de couleur Utilise le nom d'une couleur. color: red;
Hexadécimal Utilise un code couleur avec #. color: #ff0000;
rgb() Utilise rouge, vert, bleu. color: rgb(255, 0, 0);
rgba() Ajoute la transparence. color: rgba(255, 0, 0, 0.5);
background-image Ajoute une image en arrière-plan. background-image: url("image.jpg");

Boîte et espacements

Propriété Utilité Exemple
margin Ajoute un espace extérieur. margin: 20px;
padding Ajoute un espace intérieur. padding: 10px;
border Ajoute une bordure. border: 1px solid black;
width Définit la largeur. width: 200px;
height Définit la hauteur. height: 100px;
box-sizing Contrôle la taille totale d'un élément. box-sizing: border-box;

Types d'affichage

Valeur Utilité Exemple
block Prend toute la largeur disponible. display: block;
inline Reste dans la ligne. display: inline;
inline-block Reste dans la ligne mais accepte largeur et hauteur. display: inline-block;
flex Active Flexbox. display: flex;
none Cache complètement l'élément. display: none;

Flexbox

Propriété Utilité Exemple
display: flex Active le mode flex. display: flex;
flex-direction Choisit la direction des éléments. flex-direction: row;
justify-content Aligne les éléments sur l'axe principal. justify-content: center;
align-items Aligne les éléments sur l'axe secondaire. align-items: center;
gap Ajoute un espace entre les éléments. gap: 20px;
flex-wrap Permet aux éléments de passer à la ligne. flex-wrap: wrap;

Positionnement

Propriété / valeur Utilité Exemple
position: relative Déplace l'élément par rapport à sa position normale. position: relative;
position: absolute Positionne l'élément librement dans son parent. position: absolute;
position: fixed Fixe l'élément à l'écran. position: fixed;
top / right / bottom / left Déplace l'élément dans une direction. top: 10px;
z-index Gère l'ordre de superposition. z-index: 10;

Liens, listes, images et tableaux

Propriété Utilité Exemple
text-decoration Change le style des liens. text-decoration: none;
list-style Enlève ou change les puces d'une liste. list-style: none;
width: 100% Rend une image adaptable à la largeur. width: 100%;
height: auto Garde les proportions d'une image. height: auto;
border-collapse Colle les bordures d'un tableau. border-collapse: collapse;

Effets visuels

Propriété Utilité Exemple
border-radius Arrondit les coins. border-radius: 10px;
box-shadow Ajoute une ombre. box-shadow: 2px 2px 5px gray;
opacity Change la transparence. opacity: 0.7;
transition Ajoute une animation douce. transition: 0.3s;

Responsive et unités

Élément Utilité Exemple
px Unité fixe en pixels. font-size: 16px;
% Unité relative au parent. width: 50%;
vh / vw Unité relative à l'écran. height: 100vh;
em Unité relative à la taille du parent. font-size: 2em;
rem Unité relative à la racine du document. font-size: 1rem;
@media Permet d'adapter le site à différentes tailles d'écran. @media screen and (max-width: 600px)

Raccourcis utiles

But Code utile Utilité
Centrer un texte text-align: center; Centre le texte horizontalement.
Centrer avec flex display: flex; justify-content: center; align-items: center; Centre un élément horizontalement et verticalement.
Image responsive width: 100%; height: auto; Adapte l'image à la taille de l'écran.
Enlever les puces list-style: none; Retire les puces d'une liste.
Enlever le soulignement d'un lien text-decoration: none; Retire le soulignement.
Effet au survol a:hover { color: red; } Change le style quand la souris passe sur le lien.
